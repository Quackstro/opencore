# Pre-Booking Verification Checklist

## Before Searching Providers

- [ ] Patient name confirmed (as on insurance card)
- [ ] DOB verified
- [ ] Contact phone number available
- [ ] Insurance carrier and plan type known
- [ ] Member ID and Group # available
- [ ] Reason for visit documented
- [ ] Urgency level assessed (routine/urgent/emergency)
- [ ] Preferred location/ZIP codes identified
- [ ] Acceptable travel distance confirmed

## During Provider Search

- [ ] Provider accepts patient's specific plan (not just carrier)
- [ ] Provider is accepting new patients
- [ ] Provider has availability within desired timeframe
- [ ] Provider reviews checked (minimum 3.5 stars preferred)
- [ ] Office location verified within travel radius
- [ ] Office hours compatible with patient schedule

## Before Booking

- [ ] Confirm copay amount (specialist vs PCP rate)
- [ ] Check if referral required (usually not for PPO)
- [ ] Verify no pre-authorization needed for visit type
- [ ] Confirm what to bring (ID, insurance card, forms)
- [ ] Check parking availability and cost
- [ ] Note if telehealth option available for initial consult

## After Booking

- [ ] Appointment date/time confirmed
- [ ] Confirmation number saved
- [ ] Office address and directions noted
- [ ] Pre-visit paperwork identified (patient portal?)
- [ ] Reminder created (24h and 1h before)
- [ ] Brain updated with appointment details

## Red Flags - Stop and Verify

- Provider not found in insurance directory
- Unusually long wait time (>4 weeks for routine)
- No online presence or reviews
- Requires upfront payment beyond copay
- Requests SSN before appointment
