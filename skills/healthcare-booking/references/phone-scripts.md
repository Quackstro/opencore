# Phone Scripts for Healthcare Booking

## New Patient Appointment Request

```
Hi, I'd like to schedule a new patient appointment with Dr. [NAME].

My information:
- Name: [PATIENT NAME]
- Date of birth: [DOB]
- Phone: [PHONE]
- Insurance: [CARRIER] [PLAN TYPE]
- Member ID: [MEMBER_ID]
- Group: [GROUP_NUMBER]

Reason for visit: [REASON]

Do you have availability [TIMEFRAME]?
```

## Insurance Verification Call

```
Hi, I'm calling to verify that Dr. [NAME] is accepting new patients
with [CARRIER] [PLAN TYPE].

My member ID is [MEMBER_ID] and group number is [GROUP_NUMBER].

Can you confirm you're in-network for my plan?
```

## Appointment Confirmation Follow-up

```
Hi, I have an appointment scheduled with Dr. [NAME] on [DATE] at [TIME].

I wanted to confirm:
1. Is there anything I need to bring?
2. Should I arrive early to fill out paperwork?
3. Is there a patient portal where I can complete forms ahead of time?
4. What's the office address and parking situation?
```

## Questions to Ask

- [ ] Accepting new patients?
- [ ] In-network with [specific plan]?
- [ ] Next available appointment?
- [ ] Any pre-visit requirements (referral, imaging)?
- [ ] Patient portal for forms?
- [ ] Copay amount for specialist visit?
- [ ] Office address and parking?

## Common Specialist Copays (Aetna PPO typical)

- PCP visit: $20-30
- Specialist visit: $40-60
- Urgent care: $50-75
- ER: $150-300

_Always verify specific copay with insurance or practice_
